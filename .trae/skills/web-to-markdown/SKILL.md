# Web to Markdown Converter

## 功能描述

该技能用于将网页转换为Markdown格式，保留网页的主要内容结构、文本格式（包括标题层级、列表、引用等）和图片信息，同时移除不必要的广告、导航栏等非核心内容。

## 使用场景

当用户需要：
- 将网页内容转换为Markdown格式以便编辑或保存
- 保留网页的主要内容结构和格式
- 移除网页中的广告、导航栏等非核心元素
- 处理网页中的超链接，保留链接文本和URL，并提供"取消链接"选项

## 操作步骤

1. 获取目标网页的URL
2. 使用WebFetch工具获取网页内容
3. 分析网页结构，提取主要内容
4. 转换文本格式为Markdown
5. 处理超链接，保留链接文本和URL，并添加"取消链接"选项
6. 处理图片，保留图片信息
7. 移除不必要的广告、导航栏等非核心内容
8. 生成最终的Markdown文档

## 示例用法

### 输入
```
将网页 https://mp.weixin.qq.com/s/KJl2oTMaKRra2l0PV7IIiA 转换为Markdown格式
```

### 输出
```markdown
# 像玩儿一样去做事，反复强调过很多次，别做不擅长的事情

人和人的差距是很大的，虽然大家从生理上感觉差不多，但是从心智，从性格，从兴趣，从思维，从耐力，从欲望等等方面去看，那真的是天壤之别。

大家一定要明白一件事，那就是人有千千万，一定要因人而异因地制宜去做策略，就像相声行业说的，一个猴儿一个栓法。

...

**原文链接**：[像玩儿一样去做事，反复强调过很多次，别做不擅长的事情](https://mp.weixin.qq.com/s/KJl2oTMaKRra2l0PV7IIiA)（[取消链接](javascript:void(0))）
```

## 注意事项

1. 确保网页URL正确且可访问
2. 转换后的Markdown文档会保存在当前工作目录
3. 对于包含大量图片的网页，转换可能需要较长时间
4. 某些特殊格式可能无法完全转换，需要手动调整
5. 超链接的"取消链接"选项仅在支持JavaScript的Markdown编辑器中有效

## 支持的网页类型

- 新闻文章
- 博客文章
- 微信公众号文章
- 知乎回答
- 其他结构化网页内容

## 输出文件命名规则

转换后的Markdown文件将根据网页标题自动命名，格式为：`网页标题_转换时间.md`

## 质量保证

- 保留网页的主要内容和结构
- 移除广告、导航栏等非核心内容
- 转换后的Markdown文档排版清晰、结构完整
- 准确反映原网页的信息层次和内容重点
